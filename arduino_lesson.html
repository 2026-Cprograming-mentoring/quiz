CHAPTER 01

아두이노란? — Arduino UNO

아두이노는 오픈소스 마이크로컨트롤러 보드입니다. 코드를 작성하면 실제 하드웨어를 제어할 수 있는 작은 컴퓨터예요.

센서로 데이터를 읽고, LED·모터·스피커 등을 동작시킬 수 있으며, C/C++ 기반의 간단한 문법으로 프로그래밍합니다.

💡
가장 많이 사용하는 보드는 Arduino UNO이며, 이번 수업에서도 UNO를 기준으로 진행합니다.

Arduino UNO 주요 부분

디지털 핀 D0~D13
HIGH/LOW 신호 입출력
아날로그 핀 A0~A5
0~1023 값 읽기
5V / 3.3V / GND
전원 공급 핀
USB 포트
코드 업로드 & 전원
리셋 버튼
프로그램 재시작

CHAPTER 03

디지털 출력 — LED 깜빡이기

Blink.ino 
void setup() {
  pinMode(13, OUTPUT);
}

void loop() {
  digitalWrite(13, HIGH);
  delay(1000);
  digitalWrite(13, LOW);
  delay(1000);
}

핵심 함수

pinMode(핀, 모드) 핀을 INPUT 또는 OUTPUT으로 설정
digitalWrite(핀, 값) HIGH(5V) 또는 LOW(0V) 출력
delay(밀리초) 지정 시간만큼 대기 (1000=1초)
회로: 13번 핀 → 저항(220Ω) → LED(+) → GND
내장 LED로도 테스트 가능!

브레드보드 & 저항 기초

🔌 브레드보드 구조

  • ▸ 가로줄은 내부적으로 연결되어 있음
  • ▸ 상하 양쪽의 긴 줄은 전원 라인
  • ▸ 빨간줄 = VCC(+), 파란줄 = GND(–)
  • ▸ 중간 홈을 기준으로 위/아래가 분리됨
  • ▸ 납땜 없이 회로를 구성할 수 있어 편리!

⚡ 저항을 왜 사용할까?

🚫
LED에 직접 5V를 연결하면 과전류로 고장!

저항으로 전류를 제한해서 보호합니다.

220Ω → LED 보호용 (가장 흔함)
1kΩ → 일반적인 신호 제한
10kΩ → 풀업/풀다운 저항
CHAPTER 04

디지털 입력 — 버튼으로 LED 제어

Button_LED.ino 
const int btnPin = 2;
const int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(btnPin, INPUT_PULLUP);
}

void loop() {
  if (digitalRead(btnPin) == LOW) {
    digitalWrite(ledPin, HIGH);
  } else {
    digitalWrite(ledPin, LOW);
  }
}

새로운 함수

digitalRead(핀) 핀의 상태를 HIGH/LOW로 읽기

INPUT_PULLUP이란?

아두이노 내부에 풀업 저항을 활성화합니다.
→ 외부 저항 없이도 안정적 입력 가능!

버튼 안 눌림 = HIGH
버튼 눌림 = LOW

CHAPTER 05

시리얼 모니터 — 데이터 확인하기

시리얼 모니터란?

아두이노와 PC 사이의 통신 창입니다.
센서 값, 변수 등을 실시간으로 확인할 수 있고,
디버깅할 때 필수적인 도구입니다!

주요 시리얼 함수

Serial.begin(9600) 통신 시작 (보드레이트 설정)
Serial.print(값) 값 출력 (줄바꿈 없음)
Serial.println(값) 값 출력 + 줄바꿈
Serial_Hello.ino 
void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.println("Hello!");
  delay(1000);
}
CHAPTER 06

아날로그 입력 — 가변저항

디지털 vs 아날로그

디지털: 0 또는 1 (ON / OFF)
아날로그: 0 ~ 1023 (연속적인 값)

→ 밝기, 온도, 거리 등 연속적인 데이터를 읽을 때 사용!

analogRead(핀) A0~A5에서 0~1023 값 읽기
회로: 가변저항 왼쪽→5V, 가운데→A0, 오른쪽→GND
💡
가변저항을 돌리면서 시리얼 모니터에서 0~1023 값이 변하는 걸 확인해보세요!
Potentiometer.ino 
void setup() {
  Serial.begin(9600);
}

void loop() {
  int val = analogRead(A0);
  Serial.println(val);
  delay(100);
}

센서 활용 — 조도 / 온습도 / 초음파

☀️

센서 ① 조도센서 (CdS)

▼

빛의 밝기에 따라 저항값이 변하는 센서입니다.
밝으면 → 저항↓ → 값 높아짐
어두우면 → 저항↑ → 값 낮아짐

회로: 5V → CdS → A0 → 10kΩ저항 → GND
🔧
응용: 어두워지면 자동으로 LED가 켜지는 가로등 만들기! (임계값 300은 환경에 따라 조정)
CdS_LED.ino 
void setup() {
  Serial.begin(9600);
  pinMode(13, OUTPUT);
}

void loop() {
  int light = analogRead(A0);
  Serial.println(light);

  if (light < 300) {
    digitalWrite(13, HIGH);
  } else {
    digitalWrite(13, LOW);
  }
}

센서 ② 온습도센서 (DHT11)

▼
DHT11 센서 특징
온도: 0~50°C (±2°C 오차)
습도: 20~80% (±5% 오차)
3핀 연결: VCC(5V), DATA, GND
📦
라이브러리 필요! DHT sensor library (by Adafruit)

라이브러리 설치 방법

1
스케치 → 라이브러리 포함 → 라이브러리 관리
2
"DHT sensor library" 검색 (by Adafruit)
3
설치 버튼 클릭 (의존성도 함께 설치)
DHT11_Read.ino 
#include "DHT.h"
DHT dht(2, DHT11);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float t = dht.readTemperature();
  float h = dht.readHumidity();
  Serial.print(t);
  Serial.print("°C  / ");
  Serial.print(h);
  Serial.println("%");
  delay(2000);
}

센서 ③ 초음파센서 (HC-SR04)

▼

작동 원리

① Trig 핀에 10μs HIGH 신호 보냄
② 센서가 초음파 발사
③ 물체에 반사되어 돌아옴
④ Echo 핀에서 반사 시간 측정
⑤ 시간 → 거리로 변환
거리(cm) = 시간(μs) / 58.2
핀 연결: VCC→5V | GND→GND | Trig→D9 | Echo→D10
pulseIn(핀, HIGH) HIGH인 시간(μs) 측정
Ultrasonic.ino 
const int trig = 9;
const int echo = 10;

void setup() {
  Serial.begin(9600);
  pinMode(trig, OUTPUT);
  pinMode(echo, INPUT);
}

void loop() {
  digitalWrite(trig, LOW);
  delayMicroseconds(2);
  digitalWrite(trig, HIGH);
  delayMicroseconds(10);
  digitalWrite(trig, LOW);

  long dur = pulseIn(echo, HIGH);
  float dist = dur / 58.2;
  Serial.print(dist);
  Serial.println(" cm");
  delay(200);
}
