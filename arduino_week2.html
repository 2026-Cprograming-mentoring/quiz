CHAPTER 02

PWM — 아날로그 출력 흉내내기

디지털 핀은 HIGH(5V) 아니면 LOW(0V)만 가능해요. 그런데 LED 밝기를 절반으로 하고 싶다면? PWM(Pulse Width Modulation)은 아주 빠르게 껐다 켰다를 반복해서 중간 밝기/속도를 만들어내는 기술이에요.

듀티 사이클 (Duty Cycle)

100% → 항상 켜짐 = 최대 밝기
50% → 절반 켜짐 = 중간 밝기
0% → 항상 꺼짐 = 빛 없음
⚠️
PWM은 UNO에서 ~ 표시가 있는 핀만 가능: 3, 5, 6, 9, 10, 11번. 13번은 안 됩니다!
LED_Fade.ino
const int ledPin = 9; // PWM 핀!

void setup() {
  pinMode(ledPin, OUTPUT);
}

void loop() {
  // 점점 밝아지기
  for (int i = 0; i <= 255; i++) {
    analogWrite(ledPin, i);
    delay(10);
  }
  // 점점 어두워지기
  for (int i = 255; i >= 0; i--) {
    analogWrite(ledPin, i);
    delay(10);
  }
}

핵심 함수

analogWrite(핀, 0~255)PWM 출력 (0=꺼짐, 255=최대)
map(값, in_min, in_max, out_min, out_max)범위 변환
💡
analogRead()는 0~1023(10비트), analogWrite()는 0~255(8비트). 범위가 다르니까 센서 값으로 모터를 제어할 때 map() 함수로 변환하세요!
CHAPTER 03

서보모터 — 각도 제어

서보모터는 0°~180° 범위에서 원하는 각도로 회전하는 모터예요. 문 잠금장치, 로봇 팔 관절, RC카 핸들 등에 사용돼요.

🔌 핀 연결 (3핀)

빨간선 → 5V (전원)
갈색/검정선 → GND
주황/노란선 → 디지털 핀 (9번 추천)
📌
Servo 라이브러리는 아두이노에 기본 내장! 별도 설치 필요 없어요.
Servo_Basic.ino
#include <Servo.h>

Servo myServo;

void setup() {
  myServo.attach(9); // 9번 핀에 연결
}

void loop() {
  myServo.write(0);    // 0도
  delay(1000);
  myServo.write(90);   // 90도
  delay(1000);
  myServo.write(180);  // 180도
  delay(1000);
}

서보 핵심 함수

myServo.attach(핀)서보를 핀에 연결
myServo.write(각도)0~180도 회전
myServo.read()현재 각도 읽기

응용: 가변저항으로 서보 제어

Servo_Pot.ino
#include <Servo.h>
Servo myServo;

void setup() { myServo.attach(9); }

void loop() {
  int val = analogRead(A0);        // 0~1023
  int angle = map(val, 0, 1023, 0, 180); // → 0~180
  myServo.write(angle);
  delay(15);
}
💡
map(val, 0, 1023, 0, 180)이 핵심! 가변저항 값(0~1023)을 서보 각도(0~180)로 변환합니다.
CHAPTER 04

DC모터 — 속도 & 방향 제어

DC모터는 계속 회전하는 모터예요. 선풍기, 자동차 바퀴 등에 쓰여요. 서보모터와 달리 속도와 방향을 제어합니다.

🚨
아두이노 핀으로 직접 DC모터를 돌리면 안 돼요! 모터는 전류가 많이 필요해서 아두이노가 고장날 수 있어요. 반드시 모터 드라이버를 사용하세요.

L298N 모터 드라이버

모터에 충분한 전류를 공급하는 중간 다리 역할.
2개의 DC모터를 동시에 제어 가능하고,
방향 전환(정회전/역회전)도 가능해요.

🔌 L298N 핀 연결

ENA → 아두이노 PWM핀 (속도)
IN1 → 디지털 핀 (방향 1)
IN2 → 디지털 핀 (방향 2)
OUT1, OUT2 → 모터 연결
12V → 외부 전원
GND → 공통 GND (아두이노와 공유!)
IN1IN2동작
HIGHLOW정회전
LOWHIGH역회전
LOWLOW정지
DC_Motor.ino
const int ENA = 9;  // 속도 (PWM)
const int IN1 = 7;  // 방향 1
const int IN2 = 8;  // 방향 2

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void loop() {
  // 정회전 (중간 속도)
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  analogWrite(ENA, 150);
  delay(2000);

  // 역회전 (최대 속도)
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  analogWrite(ENA, 255);
  delay(2000);

  // 정지
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  delay(1000);
}
⚠️
GND를 꼭 공유하세요! 외부 전원과 아두이노의 GND를 연결하지 않으면 모터가 동작하지 않거나 불안정해요.
CHAPTER 05

피에조 부저 — 소리 출력

부저는 전기 신호를 소리로 변환하는 부품이에요. 주파수를 바꾸면 음높이가 달라지고, 지속 시간을 바꾸면 음의 길이가 달라져요.

주요 음계 주파수

도(C4)=262 · 레(D4)=294 · 미(E4)=330
파(F4)=349 · 솔(G4)=392 · 라(A4)=440
시(B4)=494 · 높은도(C5)=523
연결: + → 디지털 핀 (8번), - → GND
Buzzer_Melody.ino
const int buzzer = 8;

int melody[] = {
  262, 294, 330, 349,
  392, 440, 494, 523
};

void setup() {
  for (int i = 0; i < 8; i++) {
    tone(buzzer, melody[i], 400);
    delay(500);
  }
  noTone(buzzer);
}

void loop() { }

부저 핵심 함수

tone(핀, 주파수, 지속시간)소리 출력 (Hz, ms)
noTone(핀)소리 정지

센서 + 모터 연동 — 입력 → 출력

🎯
핵심 흐름: 센서(입력) → 조건 판단(처리) → 모터/부저(출력). 1주차 센서 + 2주차 액추에이터를 조합!
📏

초음파센서 → 서보모터

▼

거리에 따라 서보모터 각도가 변하는 예제. 손을 가까이 하면 서보가 회전해요!

Ultrasonic_Servo.ino
#include <Servo.h>
Servo myServo;
const int trig = 2, echo = 3;

void setup() {
  myServo.attach(9);
  pinMode(trig, OUTPUT);
  pinMode(echo, INPUT);
}

void loop() {
  digitalWrite(trig, LOW);  delayMicroseconds(2);
  digitalWrite(trig, HIGH); delayMicroseconds(10);
  digitalWrite(trig, LOW);
  long dur = pulseIn(echo, HIGH);
  float dist = dur / 58.2;

  int angle = map(constrain(dist,2,30), 2, 30, 0, 180);
  myServo.write(angle);
  delay(100);
}
constrain(값, 최소, 최대)값을 범위 안으로 제한
🌡️

온습도센서 → 부저 (온도 경보)

▼
Temp_Alarm.ino
#include "DHT.h"
DHT dht(2, DHT11);
const int buzzer = 8, led = 13;

void setup() {
  Serial.begin(9600);
  dht.begin();
  pinMode(led, OUTPUT);
}

void loop() {
  float t = dht.readTemperature();
  Serial.print(t); Serial.println("°C");

  if (t > 30) {
    tone(buzzer, 1000, 200);
    digitalWrite(led, HIGH);
    delay(300);
    digitalWrite(led, LOW);
  }
  delay(2000);
}

조도센서 → DC모터 (자동 팬)

▼
Light_Fan.ino
const int ENA = 9, IN1 = 7, IN2 = 8;

void setup() {
  pinMode(ENA, OUTPUT);
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
}

void loop() {
  int light = analogRead(A0);

  if (light < 300) {
    digitalWrite(IN1, HIGH);
    digitalWrite(IN2, LOW);
    int speed = map(light, 0, 300, 255, 100);
    analogWrite(ENA, speed);
  } else {
    digitalWrite(IN1, LOW);
    digitalWrite(IN2, LOW);
  }
  delay(200);
}
