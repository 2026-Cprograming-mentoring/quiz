배열의 통계값 계산 함수
난이도 중

정수 배열과 배열의 크기를 매개변수로 받아, 배열 요소의 최대값, 최소값, 평균값을 각각 포인터를 통해 반환하는 함수 getStats를 작성하세요.
평균값은 소수점 이하를 버린 정수로 계산합니다. 함수의 반환형은 void입니다.

함수 시그니처

void getStats(int arr[], int size, int *max, int *min, int *avg);
입력 예시
arr = {3, 7, 1, 9, 4, 6}
size = 6
출력 예시
max = 9, min = 1, avg = 5
solution.c

💡 힌트: 첫 번째 요소로 max, min을 초기화한 뒤, 반복문으로 비교하세요. 평균은 합계를 구한 뒤 size로 나누면 됩니다. 결과를 포인터가 가리키는 곳에 *max = 값 형태로 저장하세요.

모범 답안
void getStats(int arr[], int size, int *max, int *min, int *avg) {
    *max = arr[0];
    *min = arr[0];
    int sum = arr[0];

    for (int i = 1; i < size; i++) {
        if (arr[i] > *max) *max = arr[i];
        if (arr[i] < *min) *min = arr[i];
        sum += arr[i];
    }
    *avg = sum / size;
}
02
배열 회전 함수
난이도 중

정수 배열을 오른쪽으로 k칸 회전시키는 함수 rotateRight를 작성하세요.
예를 들어 {1,2,3,4,5}를 2칸 회전하면 {4,5,1,2,3}이 됩니다.
추가 배열(임시 배열)을 하나 사용해도 됩니다. k가 배열 크기보다 클 수 있음에 유의하세요.

함수 시그니처

void rotateRight(int arr[], int size, int k);
입력 예시
arr = {1, 2, 3, 4, 5}
size = 5, k = 2
출력 예시
arr = {4, 5, 1, 2, 3}
입력 예시 2
arr = {10, 20, 30}
size = 3, k = 5
출력 예시 2
arr = {20, 30, 10}
// k%3 = 2칸 회전
solution.c

💡 힌트: 먼저 k = k % size로 불필요한 회전을 제거하세요. 임시 배열에 결과를 만든 뒤 원본에 복사하는 방법이 가장 직관적입니다. 새 인덱스는 (i + k) % size로 계산할 수 있습니다.

모범 답안
void rotateRight(int arr[], int size, int k) {
    k = k % size;
    if (k == 0) return;

    int temp[size];

    for (int i = 0; i < size; i++) {
        temp[(i + k) % size] = arr[i];
    }
    for (int i = 0; i < size; i++) {
        arr[i] = temp[i];
    }
}
03
두 정렬 배열 병합 함수
난이도 중상

오름차순으로 정렬된 두 정수 배열 a[]와 b[]를 받아, 정렬 상태를 유지하면서 하나의 배열 result[]에 병합하는 함수를 작성하세요.
병합된 배열의 실제 원소 개수를 반환합니다. 이중 for문으로 다시 정렬하지 않고, 투 포인터 방식으로 O(n+m)에 해결해야 합니다.

함수 시그니처

int mergeSorted(int a[], int aLen,
               int b[], int bLen,
               int result[]);
입력 예시
a = {1, 3, 5, 7}
b = {2, 4, 6, 8, 10}
출력 예시
result = {1,2,3,4,5,6,7,8,10}
반환값: 9
solution.c

💡 힌트: 인덱스 i, j, k를 각각 a, b, result용으로 선언하세요. 두 배열을 앞에서부터 비교하며 작은 값을 result에 넣습니다. 한쪽이 끝나면 나머지를 그대로 복사하면 됩니다. Merge Sort의 merge 단계와 동일한 로직입니다.

모범 답안
int mergeSorted(int a[], int aLen, int b[], int bLen, int result[]) {
    int i = 0, j = 0, k = 0;

    while (i < aLen && j < bLen) {
        if (a[i] <= b[j])
            result[k++] = a[i++];
        else
            result[k++] = b[j++];
    }

    while (i < aLen)
        result[k++] = a[i++];

    while (j < bLen)
        result[k++] = b[j++];

    return k;
}

문자 배열 모음 개수 세기 함수
난이도 중상

문자 5개로 이루어진 배열을 입력받아, 입력된 문자 중 모음(a, e, i, o, u)의 개수를 세는 함수를 작성하세요.
대문자와 소문자를 모두 처리해야 합니다. (예: 'A'와 'a' 모두 모음으로 판별)
함수는 모음의 개수를 정수로 반환합니다. main 함수에서 5개의 문자를 입력받고 결과를 출력하는 전체 프로그램을 작성하세요.

함수 시그니처

int countVowels(char arr[], int size);
입력 예시
문자 5개 입력: A b e K u
출력 예시
모음 개수: 3
입력 예시 2
문자 5개 입력: x Y z W o
출력 예시 2
모음 개수: 1

요구사항

// 1. countVowels 함수를 별도로 정의할 것
// 2. 대소문자 구분 없이 a, e, i, o, u를 모음으로 판별
// 3. main에서 scanf로 5개 문자를 입력받고 함수 호출 후 결과 출력
// 4. tolower() 또는 직접 대소문자 변환 로직 사용 가능
solution.c

💡 힌트: 각 문자를 tolower()로 소문자로 변환한 뒤 'a', 'e', 'i', 'o', 'u'와 비교하세요. <ctype.h>를 include하면 tolower()를 쓸 수 있습니다. 또는 직접 if (ch >= 'A' && ch <= 'Z') ch += 32;로 변환해도 됩니다.

모범 답안
#include <stdio.h>
#include <ctype.h>

int countVowels(char arr[], int size) {
    int count = 0;
    for (int i = 0; i < size; i++) {
        char ch = tolower(arr[i]);
        if (ch == 'a' || ch == 'e' || ch == 'i' ||
            ch == 'o' || ch == 'u') {
            count++;
        }
    }
    return count;
}

int main() {
    char arr[5];

    printf("문자 5개 입력: ");
    for (int i = 0; i < 5; i++) {
        scanf(" %c", &arr[i]);
    }

    int result = countVowels(arr, 5);
    printf("모음 개수: %d\n", result);

    return 0;
}
