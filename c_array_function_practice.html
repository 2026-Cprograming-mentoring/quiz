배열의 통계값 계산 함수
난이도 중

정수 배열과 크기를 받아 최대값, 최소값, 평균값을 포인터를 통해 반환하는 함수 getStats를 작성하세요. 평균은 소수점 이하 버림.

함수 시그니처

void getStats(int arr[], int size, int *max, int *min, int *avg);
입력
arr={3,7,1,9,4,6}, size=6
출력
max=9, min=1, avg=5
입력 2
arr={-2,0,5,3}, size=4
출력 2
max=5, min=-2, avg=1
solution_1.c

💡 힌트: 첫 번째 요소로 max, min을 초기화한 뒤 반복문으로 비교하세요. 평균은 합계를 size로 나누면 됩니다. *max = 값 형태로 저장하세요.

모범 답안
void getStats(int arr[], int size, int *max, int *min, int *avg) {
    *max = arr[0];
    *min = arr[0];
    int sum = arr[0];

    for (int i = 1; i < size; i++) {
        if (arr[i] > *max) *max = arr[i];
        if (arr[i] < *min) *min = arr[i];
        sum += arr[i];
    }
    *avg = sum / size;
}

정수 배열을 오른쪽으로 k칸 회전시키는 함수 rotateRight를 작성하세요. k가 배열 크기보다 클 수 있습니다.

함수 시그니처

void rotateRight(int arr[], int size, int k);
입력
arr={1,2,3,4,5}, size=5, k=2
출력
arr={4,5,1,2,3}
입력 2
arr={10,20,30}, size=3, k=5
출력 2
arr={20,30,10}
solution_2.c

💡 힌트: k = k % size로 불필요한 회전을 제거하세요. 임시 배열에 temp[(i+k)%size] = arr[i]로 저장 후 원본에 복사하세요.

모범 답안
void rotateRight(int arr[], int size, int k) {
    k = k % size;
    if (k == 0) return;

    int temp[size];
    for (int i = 0; i < size; i++)
        temp[(i + k) % size] = arr[i];
    for (int i = 0; i < size; i++)
        arr[i] = temp[i];
}

두 정렬 배열 병합 함수
난이도 중상

오름차순 정렬된 두 정수 배열을 정렬 상태를 유지하며 result[]에 병합하세요. 투 포인터 방식 O(n+m)으로 해결해야 합니다. 원소 개수를 반환합니다.

함수 시그니처

int mergeSorted(int a[], int aLen, int b[], int bLen, int result[]);
입력
a={1,3,5,7}, b={2,4,6,8,10}
출력
result={1,2,3,4,5,6,7,8,10}
반환값: 9
입력 2
a={2,5}, b={1,3,4}
출력 2
result={1,2,3,4,5}
반환값: 5
solution_3.c

💡 힌트: 인덱스 i, j, k를 각각 a, b, result용으로 선언하세요. 앞에서부터 비교하며 작은 값을 result에 넣고, 한쪽이 끝나면 나머지를 복사합니다. (Merge Sort의 merge 단계)

모범 답안
int mergeSorted(int a[], int aLen, int b[], int bLen, int result[]) {
    int i = 0, j = 0, k = 0;

    while (i < aLen && j < bLen) {
        if (a[i] <= b[j])
            result[k++] = a[i++];
        else
            result[k++] = b[j++];
    }
    while (i < aLen) result[k++] = a[i++];
    while (j < bLen) result[k++] = b[j++];

    return k;
}

문자 배열 모음 개수 세기 함수
난이도 중상

문자 5개로 이루어진 배열에서 모음(a, e, i, o, u)의 개수를 세는 함수를 작성하세요. 대문자와 소문자를 모두 처리해야 합니다. main 함수 포함 전체 프로그램을 작성하세요.

함수 시그니처

int countVowels(char arr[], int size);

요구사항

// 1. countVowels 함수를 별도로 정의할 것
// 2. 대소문자 구분 없이 a, e, i, o, u를 모음으로 판별
// 3. main에서 scanf로 5개 문자를 입력받고 결과 출력
입력
A b e K u
출력
모음 개수: 3
입력 2
x Y z W o
출력 2
모음 개수: 1
solution_4.c

💡 힌트: tolower()로 소문자 변환 후 비교하세요. <ctype.h> include 필요. scanf(" %c", ...)에서 공백을 넣어 whitespace를 건너뛰세요.

모범 답안
#include <stdio.h>
#include <ctype.h>

int countVowels(char arr[], int size) {
    int count = 0;
    for (int i = 0; i < size; i++) {
        char ch = tolower(arr[i]);
        if (ch=='a'||ch=='e'||ch=='i'||ch=='o'||ch=='u')
            count++;
    }
    return count;
}

int main() {
    char arr[5];
    printf("문자 5개 입력: ");
    for (int i = 0; i < 5; i++)
        scanf(" %c", &arr[i]);

    printf("모음 개수: %d\n", countVowels(arr, 5));
    return 0;
}
